# Crypto Price Oracle (Web3)

This project simulates a crypto price oracle.

It helps beginners understand how smart contracts and apps get price data from outside the blockchain.

## What this project does

- Fetches token prices (simulated)
- Stores latest prices
- Shares price data to other parts of the app
- Logs price updates

## Why this project is useful

You will learn:
- What a blockchain oracle is
- How external data enters Web3 apps
- How price feeds work
- How to structure a data provider system

## Tech used

- JavaScript
- Node.js
- Oracle concepts
- Blockchain logic (simulated)

## How to run

1. Install Node.js  
2. Clone this repo  
3. Open terminal in project folder  
4. Run: `node app.js`  

## Files overview

- `app.js` → Main app file  
- `oracle.js` → Price feed logic  
- `price.js` → Token price storage  
- `feed.js` → Data sharing logic  
- `config.js` → Settings  
- `utils.js` → Helpers  
- `logger.js` → Logs  
- `network.js` → Network info  
- `sample.env` → Config example  
- `package.json` → Project info

## Notes

This project is for learning only.  
Do not use this as a real trading system.
